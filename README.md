## このプロジェクトの目的

フロントエンド技術の素振りをします

## このプロジェクトにおけるトライ

- グローバル状態管理でRecoilを使う
- Cloudflare、Vercel等のインフラを使ってみる
- css stylingをstyled-componentで行う
- Storybookを活用する
    - UI Component Setを管理する
    - Cromatic等で単体テストを書く
- React APIをより深く知る
    - ErrBoundaryやSuspenceを学ぶ
    - useContextを使う
